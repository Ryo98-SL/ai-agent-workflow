# Workbench Assets Index

## Purpose

Local image assets used by workbench UI components.

## Contents

- `deepseek.ico` is the DeepSeek provider logo.
- `openai.svg` is the OpenAI provider logo.
- `anthropic.svg` is the Anthropic provider logo.
- `ollama.png` is the Ollama provider logo.

## Constraints

Provider logos are bundled locally. UI components should not reference external
provider image URLs at runtime.
